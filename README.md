### ðŸŒŸ Array Properties & Element Checks

This section focuses on retrieving information about the array, its structure, and the nature of its elements.

### Key Concepts
* **Traversal:** Iterating over every element using a `for` or `while` loop.
* **Indexing:** Accessing elements using their position (0 to $N-1$).
* **Comparison:** Using conditional logic to find minimum, maximum, or check properties.

### 1. Basic Operations & Min/Max

### ðŸ”„ Array Restructuring & Advanced Algorithms**

This section focuses on modifying the array's content and applying complex algorithms to solve mathematical problems.

### Key Concepts
* **Shifting/Pivoting:** Moving elements to make room for insertion or deletion.
* **Pointers/Two Pointers:** Using multiple index variables (e.g., $i$ and $j$) for efficient comparisons or merging.
* **Sub-array & Pairs:** Algorithms focused on subsets of the array (like contiguous subarrays or pairs).

### 1. Insertion, Deletion & Duplicates

These files modify the array's size or content.

| File Name | Description |
| :--- | :--- |
| **Insert\_Element\_In\_Array.java** | Inserts a new value at a user-defined index. |
| **Delete\_Element\_In\_Array.java** | Deletes an element from a specific index or by value. |
| **Remove\_Duplicates\_Values\_of\_Array.java** | Removes duplicate occurrences, ensuring only unique values remain. |

### 2. Two-Array Operations & Merging

These files deal with operations involving two or more arrays.

| File Name | Description |
| :--- | :--- |
| **Merge\_two\_array.java** | Combines two arrays into a single, larger array. |
| **Intersection\_of\_Two\_Array.java** | Finds and prints the common elements (the set intersection) between two arrays. |

### 3. Complex Algorithms & Puzzles

These files implement standard and competitive programming algorithms.

| File Name | Description |
| :--- | :--- |
| **Count\_All\_pairs\_with\_Difference\_Array.java** | Counts pairs $(i, j)$ such that $|A[i] - A[j]| = k$. |
| **Distinct\_Pairs\_for\_Specific\_Diff\_Array.java** | Finds the unique pairs $(i, j)$ such that $|A[i] - A[j]| = k$. |
| **Triplets\_with\_Zero\_Sum\_Array.java** | Finds all unique triplets $(a, b, c)$ such that $a + b + c = 0$. |
| **Maximum\_Subarray\_Sum\_array.java** | Implements the maximum subarray sum problem (e.g., Kadane's algorithm). |
| **Inversion\_in\_Given\_Array.java** | Counts the number of pairs $(i, j)$ such that $i < j$ and $A[i] > A[j]$. |
| **Find\_Missing\_Value\_In\_Array.java** | Finds a single missing value in an array containing $N-1$ elements from a sequence of $N$ numbers. |
| **Missing\_No\_from1\_to\_N\_array.java** | Finds the missing number in an array of $N-1$ distinct numbers from $1$ to $N$. |

ðŸ’» Getting Started

### Prerequisites
* **Java Development Kit (JDK 8+):** Ensure you have a recent version of the JDK installed.

### Running the Code
1.  **Clone the repository:**
    ```bash
    git clone [Repository URL]
    ```
2.  **Navigate to the directory:**
    ```bash
    cd [repository-name]
    ```
3.  **Compile any Java source file** (e.g., for `Maximum_Subarray_Sum_array.java`):
    ```bash
    javac Maximum_Subarray_Sum_array.java
    ```
4.  **Execute the compiled class:**
    ```bash
    java Maximum_Subarray_Sum_array
    ```
